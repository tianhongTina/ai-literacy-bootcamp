# AI 素养训练营 - 非研发体系人员 AI 能力提升

> 科大讯飞 2026 AI 素养训练营 · 面向产品、运营、市场、HR 等非研发岗位

## 课程定位

**不是教"如何写代码"，而是教"如何用 AI 把工作做得更好、更快、更聪明"。**

- **受众**：产品经理、运营、市场、HR、行政、销售等非研发岗位
- **总课时**：8 节课，16 小时
- **形式**：场景驱动 + 动手实操 + 案例分析
- **特色**：零代码门槛，聚焦真实工作场景

## 核心理念

1. **场景优先** — 每个知识点都对应真实工作场景
2. **动手为主** — 60% 时间用于实操练习
3. **即学即用** — 当天学完，明天就能用在工作中
4. **持续迭代** — AI 工具快速演进，培养自学能力
